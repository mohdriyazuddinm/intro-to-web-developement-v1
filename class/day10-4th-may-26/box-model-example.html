CSS Box Model: content, padding, border, margin

Every element on a page is a box. The box model explains how CSS measures space around that box: the content sits in the middle, padding adds space inside the box, border wraps the padding, and margin adds space outside the box.

What each part means

  1. Content is the actual text, image, or element inside the box.
  2. Padding is the space between the content and the border. It pushes the content inward.
  3. Border is the visible line around the padding and content.
  4. Margin is the space outside the border. It pushes other elements away.

box-sizing

The box-sizing property controls whether the width and height of an element include its padding and border. The two most common values are content-box (the default) and border-box.

content-box — width excludes padding & border
160px width
border-box — width includes padding & border
160px width 
/* both boxes have the same width property
.box { width: 160px; padding: 20px; border: 8px solid #3b82f6; }

/* default */
.content-box-sample { box-sizing: content-box; /* rendered width = 160 + 2*20 + 2*8 = 216px */ }

/* recommended for predictable sizing */
.border-box-sample { box-sizing: border-box; /* rendered width = 160px (includes padding & border) */ }
