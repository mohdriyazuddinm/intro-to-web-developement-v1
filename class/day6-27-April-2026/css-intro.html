Introduction to CSS (history, versions)

CSS was introduced to solve the problem of poor styling control in early web pages.

  • In early web (1990s), HTML was only for structure, Developers started misusing HTML for styling. This created messy and hard-to-maintain code.
  • In 1994, while at CERN alongside Tim Berners-Lee, Hakon Wium Lie proposed Cascading Style Sheets (CSS) to bring typographic and layout control to HTML. The concept was later refined with Bert Bos and adopted by the World Wide Web Consortium (W3C). CSS became one of the cornerstones of modern web design, enabling flexible and consistent styling across browsers

Evolution of CSS versions

  • CSS1 (1996): The first version of CSS, introduced basic styling capabilities like fonts, colors, and margins.
  • CSS2 (1998): Added support for media types, positioning, and z-index. It also introduced the concept of the box model.
  • CSS2.1 (2011): A W3C Recommendation that fixed issues from CSS2 and represented a stable, interoperable baseline supported across browsers.
  • CSS3 era (from around 2011 onward): CSS development moved to separate modules instead of one large version release. Major features from this era include transitions, animations, media queries, Flexbox, and Grid.
  • Modern CSS (ongoing): CSS continues evolving through modules such as custom properties (CSS variables), container queries, and further layout and responsive design improvements.
  • There is no official CSS4 specification; CSS is now published as continuously updated modules.
  • CSS is now an essential part of web development, allowing developers to create visually appealing and responsive websites.
