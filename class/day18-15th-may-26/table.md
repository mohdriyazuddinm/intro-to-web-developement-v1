# table tags

- the first row is the header of the table. so use thead tag then wrap body with tbody and footer if needed with tfoot tags.

- <th> elements are not necessarily limited to being within the <thead>. They simply indicate header information. So, you can use <th> elements in the <tbody> as well if you want to indicate header information for specific rows or columns within the body of the table.

- we have rowspan, colspan attributes to merge cells in a table.

- An inappropriate use for tables is for layout. Tables should be used for tabular data, not for structuring the layout of a webpage. Using tables for layout can lead to accessibility issues and make it harder for screen readers to navigate the content. Instead, CSS should be used for layout purposes, allowing for more flexibility and better accessibility.
